We appreciate community contributions to code repositories open sourced by Microsoft. By signing a contributor license agreement, we ensure that the community is free to use your contributions.

Sign the CLA

When you contribute to a Microsoft open source project on GitHub with a new pull request, a bot will evaluate whether you have signed the CLA. If required, the bot will comment on the pull request, including a link to this system to accept the agreement.
